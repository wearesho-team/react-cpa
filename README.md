# React CPA 

Integrate [@sho-js/cpa](https://github.com/wearesho-team/cpa-frontend) with React.

## Installation
Using [npm](https://npmjs.com/package/@sho-js/react-cpa)

## Usage
To call onLoad:
```typescript jsx
import * as Cpa from "@sho-js/react-cpa";

export const Layout = () => (
    <Cpa.OnLoad />
);
```
